 Тест-кейсы для автоматизации UI (Automation Test Store)

Объект тестирования: (https://automationteststore.com/)

---

 Кейс №1: Фильтрация и сортировка в категориях
Цель: Проверить корректность работы фильтров сортировки по имени и цене.
Предусловие: Открыть главную страницу сайта.

|  | Шаг | Ожидаемый результат |
| :--- | :--- | :--- |
| 1 | Перейти в категорию "Skincare" (или любую другую, где > 4 товаров) | Открылась страница категории, отображается список товаров |
| 2 | Выбрать в выпадающем списке "Sort By" значение "Name A - Z" | Список товаров перестроился в алфавитном порядке (А -> Я) |
| 3 | Выбрать в выпадающем списке "Sort By" значение "Name Z - A" | Список товаров перестроился в обратном алфавитном порядке (Я -> А) |
| 4 | Выбрать в выпадающем списке "Sort By" значение "Price Low > High" | Товары отсортированы по возрастанию цены |
| 5 | Выбрать в выпадающем списке "Sort By" значение "Price High > Low" | Товары отсортированы по убыванию цены |

 Кейс №2: Проверка поисковой выдачи и работы с корзиной
Цель: Проверить функционал поиска, добавления позиций и пересчета суммы.

|  | Шаг | Ожидаемый результат |
| :--- | :--- | :--- |
| 1 | В поле поиска ввести "shirt" | Текст отображается в поле поиска |
| 2 | Нажать клавишу Enter | Отобразились результаты поиска по запросу "shirt" |
| 3 | Выбрать сортировку "Name A - Z" | Результаты поиска отсортированы по алфавиту |
| 4 | Добавить в корзину 2-й и 3-й товары (количество: по 2 шт каждого) | Товары добавлены, иконка корзины обновилась |
| 5 | Перейти в корзину | Открылась страница корзины с выбранными товарами |
| 6 | Найти товар с самой низкой ценой за единицу и увеличить его количество вдвое | В поле Quantity введено новое значение |
| 7 | Нажать кнопку "Update" | Количество обновлено, итоговая сумма (Total) пересчитана корректно |

---

 Кейс №3: Массовое добавление и удаление четных позиций
Цель: Проверить логику очистки корзины по порядковому номеру элемента.

|  | Шаг | Ожидаемый результат |
| :--- | :--- | :--- |
| 1 | На главной странице добавить 5 любых товаров в корзину | В корзине зафиксировано 5 различных позиций |
| 2 | Перейти в корзину | Отображается таблица с 5 выбранными товарами |
| 3 | Удалить из таблицы корзины 2-й по порядку товар | Товар исчез из списка, осталось 4 позиции |
| 4 | Удалить из обновленной таблицы 3-й по порядку товар (бывший 4-й) | Товар исчез, осталось 3 позиции (бывшие 1, 3 и 5) |
| 5 | Проверить итоговое содержимое корзины | В корзине остались только 1-й, 3-й и 5-й товары, сумма обновлена |
